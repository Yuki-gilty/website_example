# Netlify へのデプロイ方法（ドラッグ＆ドロップ）

このサイトを Netlify にドラッグ＆ドロップでデプロイする手順です。

## 手順

### 1. 静的ファイルをビルドする

ターミナルでプロジェクトのルートに移動し、次のコマンドを実行します。

```bash
npm run build
```

ビルドが完了すると、**`out`** フォルダが作成されます。この中身が Netlify にアップロードするファイルです。

### 2. Netlify にデプロイする

1. ブラウザで [https://app.netlify.com](https://app.netlify.com) を開き、Netlify にログイン（またはアカウント作成）します。
2. ダッシュボードで **「Sites」** を開き、**「Add new site」** → **「Deploy manually」** を選びます。
3. **「Drag and drop your site output folder here」** と書かれたエリアに、プロジェクト内の **`out`** フォルダ**の中身**をドラッグ＆ドロップします。
   - **注意:** `out` フォルダそのものではなく、**`out` の中のファイル・フォルダ一式**をドロップしてください（`out` を開いて、中身をすべて選択してドロップ）。
4. アップロードが終わると、サイトの URL が表示されます。クリックして表示を確認してください。

## 補足

- 更新したときは、再度 `npm run build` を実行し、新しくできた `out` の中身を Netlify の同じ場所にドラッグ＆ドロップすると再デプロイされます。
- 今後 Git リポジトリと Netlify を連携する場合は、リポジトリを Netlify に接続すると、プッシュのたびに自動でビルド・デプロイされます。その場合は `netlify.toml` の設定が使われます。
